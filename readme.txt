Jabra Perform/BlueParrott SDK - Sample Applications Package

Sample App Package Version 100.1.0

Uses SDK version 4.6.00

Includes:
- blueparrottconnectsample - an example of how to make and maintain a connection to the SDK, when the headset is connected
- blueparrottsdkaudiosample - an example of recording and  playing back audio when the BlueParrott Button is pressed
- blueparrottbatterysample - an example of how to read and monitor changes in headset battery level - does not use SDK

New sample App
- blueparrottbridge - an app that connects to the BlueParrott Button, and broadcasts android intents when the button is pressed, to be received by 3rd party apps
